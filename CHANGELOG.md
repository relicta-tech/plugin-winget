# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-12-19

### Added

- Initial release
- Winget manifest generation (version, installer, locale)
- SHA256 hash calculation for installers
- Support for x86, x64, arm, arm64 architectures
- Support for MSI, MSIX, EXE, and other installer types
- Automatic fork creation and management
- Pull request creation to winget-pkgs
- URL template support with version placeholder
- Dry-run mode for testing
- Configurable installer switches and scopes
